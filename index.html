Compilation

None of the source is compiled in a ready to use form, to do this you'll need to make sure you have npm installed.

  • git clone git@github.com:tarnfeld/jquery-switcher.git
  • cd jquery-switcher
  • make

Then open up one of the html files in the ./example folder in your browser and take a look.

Browser Support

Currently all animation is done using CSS3 transitions, and with the -webkit prefix. This will change.
